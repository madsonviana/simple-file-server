# simple-file-server
A simple file server with CORS write in Golang

### Install
```
go get -u github.com/madsonviana/simple-file-server
```

### Usage
```
simple-file-server
```

### Params
* -h 
  * Help
* -port `<port>`
  * The http port to listen (default "9999")

### Help
```
simple-file-server -h
```
